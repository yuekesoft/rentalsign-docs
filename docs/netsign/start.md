# 快速上手

## 准备

从软件开发商客服处收到`越客会议签到管理软件(网络版)`安装包。

### 安装步骤

1. 解压安装文件，得到`越客会议签到管理软件(网络版).exe`文件，双击打开进行安装。



2. 点击`下一步`，`下一步`，直到完成。
